新世纪的大门，总有些许小触动。

一、开篇


关于博客，之前有过整这些的想法，只是许久未有动作，而今，姗姗来迟。

之前有使用过Hexo配合GitHub Pages来搭建，但是有很多遗憾，所以是半成品。目前选择了WordPress来搭建，主要是因为WordPress给懒癌患者带来了很多福音，比如主题和插件，有若干的各式功能以供选择。

二、优劣性比对


但是我还是更喜欢Hexo的主题风格，怎一个美字了得。

Hexo：


  • 免费，免费，免费

  • node.js

  • Hexo + GitHub Pages 完全不需要花任何money，而WordPress则需要服务器和数据库

  • Hexo是静态博客，没有数据库，有些东西不太好管理，但是个人小博客完全胜任

  • 写博客推荐使用Cmd Markdown，免费版的Markdown，你值得拥有

  • 献给爱折腾的你最好的礼物


WordPress:

  • PHP，PHP，PHP

  • 优势便是前期折腾，后期舒爽，小白都会用

  • 博客中有媒体类资源的话还是来WP大家族吧

  • 社区更强大，你遇到的困难，总会在这里得到解决

  • 插件强大，很强大

  • 喜欢清静？来吧，来吧

三、结束语


萝卜青菜各有所爱，但是我们都爱玩技术，欢迎一起来交流。

Hello World~~
