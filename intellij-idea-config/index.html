二、常用快捷键


1、System.out.println 快捷输出


sout => System.out.println();


“abc”.sout => System.out.println(“abc”);


2、for 循环


list.for


for(String s:list){


}


3、选择一个模块：Ctrl+w


4、删除一行：Ctrl+y / Ctrl+x


5、复制选择的内容（不选择则是复制一行）：Ctrl+d


6、最近打开的文件：Crtl+e


7、当前文件搜索/整个项目搜索：Ctrl+f，Ctrl+Shift+f


8、override/implement：Ctrl+o


9、根据类名/文件名查找文件：Ctrl+n，Ctrl+Shift+n


10、跳转到指定行：Ctrl+g


11、git相关：pull：Ctrl+t，commit：Ctrl+k，push：Ctrl+Shift+K


12、万能去除疑难杂症：Alt+enter


13、导入包处理：Ctrl+Alt+o


14、添加try catch 和 折叠 等：Ctrl+Alt+t


15、括号的前或后：Ctrl+[ / Ctrl+]


16、文件内查找方法：Ctrl+f12


17、查找方法被调用的地方：


① low一点的，只显示层次：Ctrl+Alt+h（f4 进源码，Alt+8回来）


② 强大一点的，推荐：Alt+f7（PS：也可以用在变量上）


18、查找Controller方法：ctrl+alt+shift+n


19、强大的 bookmark（便于将方法保存在书签里面）：


列出：Shift+f11


改名：Ctrl+Enter


添加：在方法上 f11


20、折叠代码，比如for循环：Ctrl+Shift+.

④ Translation 翻译


⑤ Maven Helper 解决依赖冲突
